# BlueCollar Structural Steel DXF/DWG Shapes

Free structural steel DXF/DWG family packs generated from AISC v16.0 data.

## Included Families

- 2L
- C
- HP
- HSS RECT
- HSS ROUND
- L
- M
- MC
- MT
- PIPE
- S
- ST
- W
- WT

## Files

- Family DXF packs
- Family DWG packs
- Source generation scripts used in the same production cycle
- Source CSV used for generation
